前端必会知识点：fetch 和 axios

1. fetch 基本用法

fetch 是浏览器原生提供的用于发送 HTTP 请求的 API。

GET 请求示例：


fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

POST 请求示例：


fetch('https://api.example.com/data', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ name: 'John', age: 30 })
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

错误处理：

fetch 不会自动处理 HTTP 错误，需要手动检查响应状态。


fetch('https://api.example.com/data')
  .then(response => {
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }
    return response.json();
  })
  .then(data => console.log(data))
  .catch(error => console.error('Fetch error:', error));

2. axios 基本用法

axios 是一个第三方库，功能更强大，支持请求/响应拦截、请求超时等。

GET 请求示例：


axios.get('https://api.example.com/data')
  .then(response => console.log(response.data))
  .catch(error => console.error('Error:', error));

POST 请求示例：


axios.post('https://api.example.com/data', {
  name: 'John',
  age: 30
})
  .then(response => console.log(response.data))
  .catch(error => console.error('Error:', error));

错误处理：


axios.get('https://api.example.com/data')
  .then(response => console.log(response.data))
  .catch(error => {
    if (error.response) {
      console.error('Error status:', error.response.status);
    } else if (error.request) {
      console.error('No response received');
    } else {
      console.error('Axios error:', error.message);
    }
  });

3. fetch vs axios

  • axios 会自动处理 HTTP 错误，而 fetch 需要手动处理。
  • axios 默认会将响应解析为 JSON，而 fetch 需要手动调用 response.json()。
  • axios 支持请求拦截器、响应拦截器等，而 fetch 没有内建的拦截器。
  • axios 支持请求超时、取消请求等功能，fetch 需要手动实现。

4. 前端岗位常用和必会的知识

HTML & CSS

  • 了解 HTML5 语义化标签，如 <article>、<section> 等。
  • 掌握 CSS3，能够灵活使用 Flexbox 和 Grid 布局。
  • 熟悉响应式设计和常用的 CSS 预处理器（如 SCSS、LESS）。
  • 掌握常见的 CSS 动画与过渡效果。

JavaScript 基础

  • 熟悉原生 JavaScript（ES6+），如箭头函数、解构赋值、模板字符串等。
  • 了解异步编程，能够熟练使用 Promise、async/await。
  • 理解闭包、作用域链等概念，能够处理常见的 JS 问题。

前端框架与库

  • 掌握至少一种现代 JavaScript 框架，如 React、Vue、Angular。
  • 了解如何使用 Redux 或 Vuex 等状态管理工具。
  • 理解组件化开发，并能使用组件库，如 Ant Design、Element UI。

工具与技术

  • 熟悉 Git 版本控制，能够进行常见的 Git 操作。
  • 了解前端构建工具，如 Webpack、Vite。
  • 掌握调试工具，如 Chrome DevTools，能够进行性能分析。

其他必备技能

  • 具备一定的 Node.js 后端知识，能够处理基础的后端请求。
  • 熟悉 RESTful API 设计原则，能够进行前后端数据交互。
  • 了解基本的 SEO 知识，能够优化页面加载速度和搜索引擎排名。
