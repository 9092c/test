Git 速查表

快速查看常用的 Git 命令和操作

基础操作

git init — 初始化一个 Git 仓库

git clone [url] — 克隆一个远程仓库

git status — 查看当前工作区状态

git add [file] — 将文件添加到暂存区

git commit -m "message" — 提交更改并添加提交信息

git log — 查看提交历史

分支操作

git branch — 查看所有分支

git branch [branch-name] — 创建新分支

git checkout [branch-name] — 切换分支

git merge [branch-name] — 合并分支

远程操作

git remote -v — 查看远程仓库

git push [origin] [branch-name] — 将分支推送到远程仓库

git pull [origin] [branch-name] — 从远程仓库拉取更新

git fetch — 获取远程仓库更新

其他常用操作

git diff — 查看当前更改

git stash — 暂存当前更改（撤销更改但保留）

git reset [file] — 撤销文件的更改

git rm [file] — 删除文件并从版本控制中移除

git tag [tag-name] — 为提交打标签

小贴士

建议使用 Git 分支：在开发新功能或修复 bug 时，始终创建一个新的分支。这样可以保持主分支（如 `master` 或 `main`）的稳定性。
