基础知识与工作中常用知识点

1. 变量声明

JavaScript 中的变量可以使用 var, let 和 const 来声明。

let name = 'John'; // 使用 let 声明变量
const age = 25; // 使用 const 声明常量
var isStudent = true; // 使用 var 声明变量

let 和 const 是 ES6 引入的，推荐在现代 JavaScript 中使用。var 声明的变量有一些不易察觉的特点，通常不推荐使用。

2. 函数

函数是可重用的代码块，通过调用函数可以执行某个任务。

function greet(name) {
    return 'Hello, ' + name;
}
console.log(greet('Alice'));

函数是 JavaScript 中最基本的结构之一，允许你定义和复用代码块。函数可以接收参数并返回值。

3. 数组

数组是用于存储多个数据的容器，可以通过索引来访问。

let fruits = ['Apple', 'Banana', 'Orange'];
console.log(fruits[0]); // 输出 Apple

数组是 JavaScript 中常用的数据结构，允许你存储多个值。你可以通过索引访问数组中的元素，索引从 0 开始。

4. 对象

对象是存储键值对的数据结构，用于表示复杂数据。

let person = {
    name: 'Alice',
    age: 30,
    greet: function() { return 'Hello, ' + this.name; }
};
console.log(person.name);

对象是 JavaScript 中用于组织和存储数据的一种重要数据结构。你可以通过键（属性）来访问对象中的值。

5. 异步编程

JavaScript 处理异步操作时，可以使用回调函数、Promises 或者 async/await。

// 使用 Promise
let promise = new Promise((resolve, reject) => {
    if (true) {
        resolve('成功！');
    } else {
        reject('失败！');
    }
});
promise.then(result => console.log(result)).catch(error => console.log(error));

异步编程允许 JavaScript 在等待某些操作时（如网络请求）继续执行其他任务。Promise 是处理异步操作的一种方法，而 async/await 则是更简洁的语法。

6. ES6 新特性

ES6 引入了很多新特性，提升了 JavaScript 编程的简洁性与灵活性。

// 模板字符串
let name = 'Alice';
let greeting = `Hello, ${name}!`; // 使用模板字符串
console.log(greeting);

// 解构赋值
let [a, b] = [1, 2];
let {name, age} = {name: 'Alice', age: 30};
console.log(a, b, name, age);

ES6 为 JavaScript 带来了很多便捷的语法和特性，比如模板字符串、解构赋值、箭头函数、类、模块化等。这些特性可以极大地提升开发效率和代码可读性。

7. 浏览器 API

浏览器提供了很多 API 让 JavaScript 与浏览器进行交互，常用的有 Web Storage API、Fetch API 等。

// 使用 Fetch API 获取数据
fetch('https://jsonplaceholder.typicode.com/posts')
    .then(response => response.json())
    .then(data => console.log(data));

// 使用 localStorage 保存数据
localStorage.setItem('username', 'Alice');
console.log(localStorage.getItem('username'));

浏览器提供了多种 API 让 JavaScript 操作浏览器内的功能，比如通过 Fetch API 发起网络请求，使用 Web Storage API 存储数据等。
